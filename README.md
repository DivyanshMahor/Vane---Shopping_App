## 📖 Overview

**Vané** is a luxury cosmetics shopping app that brings a sleek, modern, and elegant UI to product discovery and browsing. Built with Flutter and powered by Riverpod for state management, it fetches live product data from the DummyJSON API and presents it through a polished, responsive interface.

---

## ✨ Features

- 🛍️ Product listing fetched from the DummyJSON API
- 🔍 Detailed product view with full information
- 🎨 Clean, modern, luxury-themed UI
- ⚡ Riverpod for predictable and scalable state management
- 🌐 Dio for fast and reliable API handling
- 📱 Fully responsive design across screen sizes

## 🚀 Getting Started

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed
- An IDE such as VS Code or Android Studio

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/<your-username>/vane.git

# 2. Navigate into the project directory
cd vane

# 3. Install dependencies
flutter pub get

# 4. Run the app
flutter run
```
